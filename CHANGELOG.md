# Changelog

## [1.0.0] - 2026-04-14

### Added
- Semantic property-set duplicate detection (order-insensitive, min 2 properties)
- Full LESS/SCSS nesting support with `&` parent references and absolute line numbers
- Safe merge with VS Code diff preview — nothing auto-applied
- Per-rule Diff and Delete buttons in results panel
- Rich webview results panel with collapsible group cards
- Expand All / Collapse All controls
- Quick Fix code actions (Merge, Dismiss)
- Workspace, folder, and single-file scanning
- Configurable severity levels and exclude filters
- Status bar indicator with quick-pick menu
- Auto-scan on save and editor change
- 209 tests covering parser, scanner, merger, edge cases, and integration
